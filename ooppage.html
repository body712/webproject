Introduction to Object-Oriented Programming (OOP)

What is Object-Oriented Programming?

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which are instances of classes. OOP allows the structuring of software in a more modular and reusable way.

Core Principles of OOP

OOP is built on the following core principles:

  • Encapsulation: Bundling of data and methods that operate on the data.
  • Inheritance: Mechanism where one class can inherit properties and behaviors from another.
  • Polymorphism: Ability to take multiple forms, enabling the same method to behave differently based on the object.
  • Abstraction: Hiding the complex implementation details and exposing only the necessary parts.

Class and Object

In OOP, a class is a blueprint for creating objects. An object is an instance of a class. Here’s a simple example: 

class Car {
  constructor(make, model) {
    this.make = make;
    this.model = model;
  }

  drive() {
    console.log(\`Driving a \${this.make} \${this.model}\`);
  }
}

const myCar = new Car('Toyota', 'Corolla');
myCar.drive();  // Output: Driving a Toyota Corolla

Inheritance in OOP

Inheritance allows a class to inherit properties and methods from another class. It promotes reusability and a hierarchical class structure. 

class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(\`My name is \${this.name}\`);
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name);
    this.breed = breed;
  }

  speak() {
    console.log(\`My name is \${this.name}, and I am a \${this.breed} dog\`);
  }
}

const myDog = new Dog('Buddy', 'Golden Retriever');
myDog.speak();  // Output: My name is Buddy, and I am a Golden Retriever dog

Polymorphism and Abstraction

Polymorphism allows different classes to define methods with the same name, but with different implementations. Abstraction hides unnecessary implementation details. 

class Shape {
  area() {
    throw "Area method must be implemented in subclass";
  }
}

class Circle extends Shape {
  constructor(radius) {
    super();
    this.radius = radius;
  }

  area() {
    return Math.PI * Math.pow(this.radius, 2);
  }
}

const myCircle = new Circle(5);
console.log(myCircle.area());  // Output: 78.53981633974483

Real-World Applications of OOP

  • Game Development: OOP is widely used for creating game objects like characters, enemies, and weapons.
  • Software Applications: Most modern applications use OOP for better code organization and maintainability.
  • Web Development: JavaScript frameworks and libraries like React use OOP principles.
